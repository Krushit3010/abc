
# Formbuilder  V-1.0.0
## Features

- Create Dynamic Forms
- User Can Save and Download Form
- User Can See Recent Created Forms
- User Can Edit Created Forms
- Controls With Customization
    - File Upload
    - Number
    - Header
    - Text
    - Textarea
    - Radio group
    - Checkbox
    - Datetime
    - Select
    - Geometry
    - Auto Count
    - Star Rating
    - Range Field
    - Signature
    - QR Code
    - Separator
- Dependent Controls
    - Create Dependent on Select / Checkbox / Radio-group
- Download & Upload(Edit) IQF File
- Save & Load Locally Store Form
- Preview Form
